### Freshcoins Bot

Automated vote bot for `Freshcoins.io` built with TypeScript and Puppeteer.

#### Features
- Supports both authenticated and unauthenticated proxies
- Rotates proxy after each vote session

#### Setup
```bash
npx tsx src/main.ts
```
