### Freshcoins Bot

This is an automated vote bot for FreshCoins.io built with TypeScript and Puppeteer.

#### Features
- Supports both authenticated and unauthenticated proxies
- Rotates proxy after each vote session

#### Setup
```bash
npx tsx src/main.ts
```
or
```bash
npx tsx main.ts
```
